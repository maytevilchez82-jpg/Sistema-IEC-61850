# Sistema IEC 61850

Proyecto estático de visualización y exportación de estructuras IEC 61850.

## Estructura del proyecto

- `index.html`
- `css/style.css`
- `js/ln-selection-optimizer.js`
- `js/performance-optimizer.js`
- `js/script-rapido.js`
- `Nakama Logo.jpeg`

## Uso

1. Abrir `index.html` en el navegador.
2. Subir un archivo `.cid`, `.icd`, `.scd`, `.ssd` o `.xml`.
3. Navegar el árbol y exportar datos según sea necesario.

## Nota

Este repositorio es un sitio estático sin servidor; se puede publicar en Netlify, GitHub Pages o cualquier hosting de archivos estáticos.
